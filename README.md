# Air_Quality_Station
This DIY air quality monitoring system integrates precision sensors with a connected processing platform and a low-power e paper display.  it monitors :  temperature, pressure, humidity, CO2 level and air quality (particules/dust).

Full project is described on my hackaday pages : https://hackaday.io/project/205888-air-quality-station

Esp32-S2 source code is here : https://github.com/f2knpw/Air_Quality_Station/blob/master/air_quality.ino
